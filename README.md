A Java-based event management system with user registration, venue browsing, booking creation, and admin approval workflows. It uses Servlets, JDBC, and a relational database to manage users, locations, bookings, and payment records. The system includes an admin dashboard for reviewing and approving bookings, plus a payment module that stores transaction details and approval proof. Front-end pages are built with HTML, CSS, and JavaScript to provide booking, payment, and dashboard interactions. The project demonstrates full-stack development, database design, and workflow automation for event booking and payment processing.
